# A股量化研究平台

独立的 A 股量化股票库，沿用个人交易成长日志的 UI。网站使用 Supabase Auth、独立量化数据表和 Edge Functions；Yahoo Finance 提供日 K 线，东方财富公开接口提供 A 股名单及免费资金流排名。

## 功能

- 使用原 Supabase 账号登录
- 导入全 A 股并独立保存
- 每批扫描 20 只股票
- 全市场名单按100只一页分页读取，避免公开接口单页上限造成漏股
- Yahoo Finance 失败时自动切换东方财富免费日 K 线
- MA5/10/20/60、RSI14、动量、量比、波动率评分
- 同步全市场今日、5 日和 10 日主力资金净流入排名
- 计算 OBV、MFI、CMF、VPT 等量价代理指标
- 技术面 55% + 免费资金面 45% 的综合观察分
- K 线与均线图
- 搜索并筛选综合强势、资金偏强和综合风险股票
- 显示每批成功数、失败数和行情错误原因

免费“主力资金”是按公开成交单分类推算的统计口径，无法识别具体机构或量化基金，也不是实时持仓。所有评分仅用于缩小研究范围，不等于完整投资价值，不构成投资建议。
